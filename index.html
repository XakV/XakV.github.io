Same thing we do every night Pinky, reset our ZNC password

Published Thu 08 November 2018 in Posts

by Zach Villers Linux Firewall libvirt

If I write it down somewhere, I'll remember it. Why don't I do that with passwords?

~right~

For the next time you forget your znc password, here is how to reset it.

    [me@abox]$ znc --makepass
    [me@abox]$ # this returns a hash and a salted password
    [me@abox]$ vi /var/run/znc/.znc/configs/znc.conf
    [me@abox]$ # then copy/paste in the new password and hash
    [me@abox]$ # it should be at the bottom of the config file

Easy, Peasey, and Lemon-Squeezey.
